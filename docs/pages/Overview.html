Overview

Here's the list of controls and components, and their intended purposes!

Controls

Control Description
cuiButton A button with customizable colors, optional Image, and roundable corners. Its CheckButton and Checked properties allow it to behave like a toggle button.
cuiButtonGroup A radio button with customizable colors, optional Image, and roundable corners. Its Group property lets you distinguish different radio button groups inside the same parent.
cuiCalendarDatePicker A custom button that displays a calendar icon alongside the selected date. Clicking it opens a customizable date selection dialog.
cuiCheckbox A checkbox with adjustable colors, roundable corners, and optional checked/unchecked symbols.
cuiColorPickerWheel A highly-optimized HSV triangle color picker featuring an outer hue ring and an inner saturation/value triangle.
cuiComboBox A modern ComboBox that displays items in a rounded dialog with a matching custom scrollbar when needed.
cuiFileDropper Accepts one or more dropped files with optional extension filtering, or opens an OpenFileDialog when clicked.
cuiFolderDropper Accepts one or more dropped folders or opens the modern OpenFolderDialog when clicked.
cuiGroupBox A panel-like container with customizable colors and rounded corners.
cuiLabel A label whose text can be aligned both vertically and horizontally.
cuiListbox A list box with customizable colors and optional rounded corners for both the control and its items.
cuiOTPasswordBox A one-time password input control. The BoxAmount property controls the number of input boxes.
cuiPanel A panel with customizable colors and rounded corners.
cuiPanelGradient A panel with customizable rounded corners and gradient background and border.
cuiPictureBox A picture box supporting image tinting, rotation, and rounded corners via the ImageTint, Rotate, and Rounding properties.
cuiCircleProgressBar A circular progress bar with configurable thickness and optional rounded ends.
cuiProgressBarHorizontal A horizontal progress bar with customizable colors, shape, direction, and corner rounding.
cuiProgressBarVertical A vertical progress bar with customizable colors, shape, direction, and corner rounding.
cuiProgressTrackerHorizontal A horizontal progress tracker for displaying progress through multiple steps or tasks.
cuiProgressTrackerVertical A vertical progress tracker for displaying progress through multiple steps or tasks.
cuiResizeGrip A resize grip for borderless windows. Recommended for use with the cuiFormRounder component.
cuiSeparator A divider control with configurable color, thickness, orientation, and margin.
cuiSlider A customizable horizontal slider/trackbar.
cuiSliderVertical A customizable vertical slider/trackbar.
cuiSpinner An animated loading spinner for indicating background activity.
cuiStarRating A star rating control supporting half-star increments. The maximum Rating value is twice the StarCount.
cuiSwitch An animated toggle switch with customizable colors.
cuiTabControl A custom TabControl whose pages are created programmatically using the AddTab(...) methods instead of the designer.
cuiTextBox A text box with customizable colors, rounded corners, optional multiline support, and an optional tintable image.
cuiTextBoxCreditCardNumber A specialized text box that formats credit card numbers in groups of four digits and automatically detects the card provider.
cuiEllipse A simple ellipse-shaped control with a customizable color.
cuiHexagon A hexagon-shaped control with customizable colors and rounded corners.
cuiChartLine A single-line chart with customizable colors and optional Bézier curve smoothing.

Components

Component Description
cuiControlAnimator This component animates a control's position and can also animate the opacity of most controls by attaching to their OnPaint event. The EasingType property lets you choose from easing styles such as Quart Out and Back In-Out.
cuiControlBlur Applies a highly optimized blur effect to the target control by attaching to its OnPaint event.
cuiControlDrag Allows a control to be dragged with the mouse to move the form window. Particularly useful for creating custom title bars in borderless forms.
cuiFormAnimator Animates the target form's Opacity property over a configurable duration using the selected easing type.
cuiFormDrag Allows the user to move a form window by dragging anywhere on the form. Useful for borderless applications.
cuiFormHideCaptureScreenshot Prevents the form from appearing in screenshots and screen recordings. The ExclusionType property controls whether the window appears black or completely invisible.
cuiFormRounder Rounds the corners of a borderless form. Only one cuiFormRounder can be assigned per form. This component is also used internally by HartUI dialogs, including those created by cuiMessageDialog.
cuiMessageDialog Dims the parent form and displays a customizable message dialog on top. The dialog's colors, corner radius, size, and padding can all be adjusted.
cuiOpenFolderDialog Provides easy access to the modern OpenFolderDialog as a replacement for the legacy FolderBrowserDialog.
cuiTaskbarBadgeController Adds a notification badge to the application's taskbar icon or displays a custom badge image.
cuiTaskbarStateController Controls the application's taskbar state, including Default, Error, Paused, and Progress, and allows the progress value to be updated.
cuiTooltipHover Displays a customizable tooltip when the mouse pointer hovers over the target control.
