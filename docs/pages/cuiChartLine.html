cuiChartLine

The cuiChartLine is a basic chart control.
It lets you visualise one fluctuating value at a time.

The Y axis can be represented either with integers or percentages.
The X axis are week days by default, but they are customizable through the CustomXAxis property.

Highlighted Properties

The control exposes many properties, these are the most important ones.

Color

AxisColor

Specifies the colour of the vertical and horizontal axes of the chart.


Color

ChartLineColor

Specifies the colour of the main line.


bool

GradientBackground

Controls whether the area under the main line will display as a gradient or a solid colour.


Color

PointColor

On each point, a circle will display with this colour.
If you wish to remove the points, just set PointColor to Color.Empty.
