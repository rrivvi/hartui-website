cuiButtonGroup

The cuiButtonGroup is the same as cuiButton, except it introduces Group-based radio behaviour.
There's a similar control: cuiButton, without the radio behaviour.

When clicked, it will uncheck all other group buttons who share the same Parent and Group value, and become checked itself.

Highlighted Properties

The control exposes many properties, these are the most important ones.

int

Group

When clicked, other group buttons from the same Parent with the same Group value will become unchecked.


Padding

Rounding

Specifies how rounded the corners should be, in pixels.


String

Content

The text content of the control.
For newcomers, you can also use the Text variable in code, but it's not available from the Designer.


bool

Checked

This property will determine if the button is in a checked state.
The checked state has priority above mouse states.

When set to true, other group buttons from the same Parent with the same Group value will become unchecked.

Image

Image

Your Image will be displayed to the left of the Content

Its Size can be manually adjusted with the ImageExpand property.
Its Position relative to the text Content can be adjusted with the TextSpacing property.

Exposed Methods

Winforms controls expose many controls by default, here are the most important ones.

void

PerformClick()

Runs mouse click logic, then invokes the OnClick EventHandler.
