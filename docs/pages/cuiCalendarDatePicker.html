cuiCalendarDatePicker

The cuiCalendarDatePicker is a simple button that, when pressed, will open CuoreUI's custom date picker popup.

Highlighted Properties

The control exposes many properties, these are the most important ones.

Padding

Rounding

Specifies how rounded the corners should be, in pixels.


String

Content

The text content of the control.


bool

CheckButton

Controls whether the control can become checked with a mouse click.


Image

Icon

Your Icon will be displayed to the left of the centered Content.


CuoreUI.Controls.Forms.DatePicker.Themes

Theme

Set the popup form's theme to either Light or Dark


bool

EnableThemeChangeButton

If True, the user will be able to change the popup's theme, by pressing a button inside the popup form.
