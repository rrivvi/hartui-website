OpenFolderDialog

A modern replacement for FolderBrowserDialog that uses the Windows File Open dialog in folder selection mode. It supports modern UI, multi-selection, and several customization options.

Important Properties

Property Description
Multiselect Allows the user to select multiple folders.
FolderName Gets the first selected folder.
FolderNames Gets all selected folders.
InputPath Specifies the initial folder displayed when the dialog opens.
Title Sets the dialog window title.
OkButtonLabel Changes the text displayed on the confirmation button.
FileNameLabel Changes the label displayed next to the folder name field.

Methods

Method Description
ShowDialog() Displays the dialog using the application's main window as its owner.
ShowDialog(IntPtr) Displays the dialog using the specified window handle as its owner.
Tip
This class is public so it can be used independently of cuiOpenFolderDialog. It provides the modern Windows folder picker on both .NET Framework and .NET.

Note
FolderNames always contains every selected folder, while FolderName returns only the first one.
