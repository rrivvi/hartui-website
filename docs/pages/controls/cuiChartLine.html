cuiChartLine

Displays a customizable single-line chart with optional Bézier curve smoothing, gradient fill, and point value tooltips.

Important Properties

Property Description
DataPoints The values displayed on the chart. At least two data points are recommended for proper rendering.
CustomXAxis Replaces the default weekday labels with custom X-axis labels. Leave empty to use the built-in weekday labels.
AutoMaxValue Automatically sets MaxValue to the largest value in DataPoints. Enabled by default.
MaxValue Sets the maximum value displayed on the Y-axis. Assigning this property automatically disables AutoMaxValue.
UsePercent Controls whether Y-axis labels and point tooltips display values as percentages or absolute numbers.
ShortDates Uses abbreviated weekday names (Mon, Tue, ...) instead of full names when the default X-axis labels are used.
UseBezier Draws smooth Bézier curves instead of straight line segments.
GradientBackground Draws a gradient fill beneath the chart line. When disabled, a solid translucent fill is used instead.
ShowLines Controls whether horizontal guide lines are displayed.
ChartPadding Specifies the padding around the chart drawing area.
Tip
Hover the mouse over any data point to display its exact value in a popup.

Caution
CustomXAxis must contain either zero labels or at least two. Supplying exactly one label is not supported.

Note
Setting MaxValue automatically disables AutoMaxValue. Re-enable AutoMaxValue if you want the chart to resume scaling itself based on the data.
