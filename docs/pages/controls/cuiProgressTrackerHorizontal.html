cuiProgressTrackerHorizontal

Displays progress through a sequence of tasks using connected indicators. Completed, current, and remaining tasks are shown with different visual states.

Important Properties

Property Description
Tasks The list of task names displayed by the control.
TasksProgress Specifies how many tasks have been completed.
CurrentTask Read-only. Returns the name of the current task.
ShowSymbols Determines whether completed tasks display a checkmark.
AutoRounding Automatically rounds each task indicator into a circle. Disable this to use the Rounding property.
Caution
The control requires at least two tasks. If fewer are provided, nothing is rendered.

Tip
Set TasksProgress to 1 to highlight the first task as the current step. Increasing the value marks previous tasks as completed automatically.
