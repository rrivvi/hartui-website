cuiStarRating

Displays a row of stars for selecting or displaying a rating. Supports half-star increments and can optionally be made read-only.

Important Properties

Property Description
StarCount Controls how many stars are displayed.
Rating Gets or sets the current rating. The maximum value is StarCount × 2, allowing half-star increments.
Rounding Controls how rounded the star points are. Higher values make the stars more rounded, while lower values less rounded. Rounding = 0 makes the corners sharp.
AllowUserInteraction Determines whether the user can change the rating with the mouse.

Events

Event Description
RatingChanged Occurs whenever the Rating property changes.
Tip
Ratings use half-star precision. For example, a value of 7 represents 3.5 stars when StarCount is 5.
