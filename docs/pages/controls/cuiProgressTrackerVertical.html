cuiProgressTrackerVertical

Displays progress through a sequence of tasks using a vertical step tracker. Completed tasks, the current task, and remaining tasks are visually distinguished.

Important Properties

Property Description
Tasks Specifies the list of task names displayed by the control.
TasksProgress Determines how many tasks have been completed.
CurrentTask Read-only. Returns the name of the current task.
ShowSymbols Determines whether completed tasks display a checkmark.
LineThickness Controls the thickness of the connecting lines.
AutoRounding Automatically rounds task indicators into circles. Disable this to use the Rounding property.
Rounding Controls the corner radius of the task indicators when AutoRounding is disabled.
Caution
The control requires at least two tasks. If fewer than two are provided, nothing is rendered.

Tip
TasksProgress represents the number of completed tasks. The next task in the sequence is automatically highlighted as the current task.
