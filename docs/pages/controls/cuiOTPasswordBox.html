cuiOTPasswordBox

A one-time password input control that displays each character in its own box and automatically tracks when all boxes have been filled.

Important Properties

Property Description
BoxAmount Determines the number of character boxes displayed.
Content Gets or sets the entered text. Text is an alias for this property.
OnlyDigit Intended to restrict input to digits only.
UnderlinedStyle Draws only the bottom border of each box instead of a full outline.

Methods

Method Description
ResetText() Clears the current input.

Events

Event Description
FinishedTypingContent Occurs when the number of entered characters reaches BoxAmount.
NotFinishedTypingContent Occurs when the content becomes shorter than BoxAmount, such as after deleting a character.
Tip
The control supports pasting text with Ctrl+V. Characters are inserted until all boxes are filled.

Caution
Input is automatically limited to letters and digits, and all entered letters are converted to uppercase.
