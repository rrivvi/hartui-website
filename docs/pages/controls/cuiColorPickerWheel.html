cuiColorPickerWheel

An HSV color picker consisting of an outer hue ring and an inner saturation/value triangle.

Important Properties

Property Description
Content Gets or sets the currently selected color.
WheelThickness Controls the thickness of the hue ring. Increasing this value reduces the size of the saturation/value triangle.

Events

Event Description
ContentChanged Occurs whenever the selected color changes, either by user interaction or by assigning Content from code.
SelectedColor Occurs after the user finishes selecting a color by releasing the left mouse button.
Tip
Use ContentChanged for live previews while the user is dragging the selection, and SelectedColor when you only need the final color.

Note
Assigning Content automatically updates the wheel and selection indicator to match the specified color.
