cuiSliderVertical

A customizable vertical slider for selecting a value within a specified range by dragging the thumb.

Important Properties

Property Description
Value Gets or sets the current slider value.
MinValue Specifies the minimum value the slider can represent.
MaxValue Specifies the maximum value the slider can represent.
UpsideDown Reverses the slider direction so higher values appear at the top instead of the bottom.
ThumbOutlineThickness Controls the thickness of the outline drawn around the slider thumb.

Methods

Method Description
GetProgressPercentage() Returns the current slider position as a value between 0 and 1.

Events

Event Description
ValueChanged Occurs whenever the Value property changes.
Tip
Enable UpsideDown when you want the slider to behave like a traditional vertical volume or progress control, where larger values are positioned at the top.

Caution
MinValue must remain smaller than MaxValue. Values assigned outside the valid range are ignored.
