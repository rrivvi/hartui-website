cuiTextBoxCreditCardNumber

A specialized text box that automatically formats credit card numbers, detects the card provider, and displays the corresponding card logo.

Important Properties

Property Description
Content Gets or sets the current credit card number. This property is equivalent to Text.
PlaceholderText Text displayed when the control is empty and does not have focus.
Company Gets the detected card provider.
IsCardValid Indicates whether the entered number is long enough to identify a supported card provider.
PasswordChar Displays password characters instead of the entered text.
UnderlinedStyle Draws only the bottom border instead of a complete outline.
Rounding Controls the corner radius of the text box.

Methods

Method Description
ResetText() Clears the current text and restores the placeholder if applicable.
DetectCardType(string) Detects the card provider from the specified card number.

Events

Event Description
ContentChanged Occurs whenever the entered card number changes.
IsCardValidChanged Occurs when the value of IsCardValid changes.
Tip
The card logo updates automatically as the user types. The detected provider can also be retrieved through the Company property.

Caution
IsCardValid only verifies that the entered number matches the length and prefix of a supported card provider. It does not perform a checksum (Luhn) validation.
