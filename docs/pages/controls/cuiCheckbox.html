cuiCheckbox

A customizable checkbox control with rounded corners, configurable colors, and optional checkmark and cross symbols.

Important Properties

Property Description
Checked Gets or sets whether the checkbox is checked.
Content The text displayed next to the checkbox. Text is provided as an alias for this property.
ShowSymbols Determines whether the checkbox displays a checkmark when checked and a cross when unchecked.
Rounding Controls the corner radius of the checkbox.
OutlineThickness Controls the thickness of the checkbox outline.

Events

Event Description
CheckedChanged Occurs whenever the Checked property changes.
Tip
The control overrides the standard Text property, making it an alias of Content. Either property can be used to set the displayed text.

Note
Clicking the control automatically toggles the Checked property and raises the CheckedChanged event.
