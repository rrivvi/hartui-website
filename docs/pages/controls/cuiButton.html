cuiButton

A customizable button supporting rounded corners, images, toggle behavior, and independent appearance settings for its normal, hovered, pressed, and checked states.

Important Properties

Property Description
Content The text displayed by the button. This property is also exposed through Text.
DialogResult Sets the parent form's DialogResult automatically when the button is clicked.
CheckButton Enables toggle button behavior using the Checked property.
Checked Gets or sets whether the button is currently checked. Changing this property raises the CheckedChanged event.
Image Displays an image next to the button text.
ImageExpand Adjusts the size of the displayed image. By default, the image is rendered using the current font height.
TextAlignment Aligns the button content to the left, center, or right using Near, Center, or Far.
TextSpacing Controls the spacing between the image and text.
TextPadding Adds padding around the button content. This property is most useful when TextAlignment is Near or Far.
Rounding Controls the corner radius of the button.
OutlineThickness Specifies the thickness of the button outline.

Events

Event Description
CheckedChanged Occurs whenever the Checked property changes.
Tip
The button exposes separate colors for the background, outline, text, and image tint for each visual state, allowing every state to be customized independently.

Caution
The Checked property only changes automatically when CheckButton is enabled. Otherwise, the button behaves like a standard push button.

Note
Setting DialogResult automatically updates the parent form's DialogResult when the button is clicked, making the control suitable for dialog windows without requiring additional code.
