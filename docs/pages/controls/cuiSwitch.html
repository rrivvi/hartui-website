cuiSwitch

An animated toggle switch that provides a modern alternative to a traditional checkbox.

Important Properties

Property Description
Checked Gets or sets whether the switch is in the checked state.
OutlineThickness Controls the thickness of the switch's outline.
ThumbSizeModifier Adjusts the size of the thumb without changing the overall control size.

Events

Event Description
CheckedChanged Occurs whenever the Checked property changes.
Tip
Clicking the switch automatically toggles the Checked property and plays the built-in thumb animation.
