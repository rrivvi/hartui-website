cuiHexagon

Displays a customizable hexagon with configurable corner rounding, fill color, and optional outline.

Important Properties

Property Description
PanelColor Specifies the fill color of the hexagon.
OutlineColor Specifies the color of the hexagon outline.
OutlineThickness Controls the thickness of the outline in pixels.
Rounding Controls how rounded the six corners of the hexagon appear. Higher values produce smoother corners.
Note
The hexagon automatically scales to fit the control while preserving its shape and respecting the configured outline thickness.
