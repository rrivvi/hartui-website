cuiFileDropper

Accepts files by drag and drop or, optionally, by opening an OpenFileDialog when clicked.

Important Properties

Property Description
UploadWithClick Determines whether clicking the control opens an OpenFileDialog.
Multiselect Allows the user to select or drop multiple files at once.
Filter Specifies the allowed file types. The same filter is applied to both drag-and-drop and the file dialog.
FileName Gets the first selected or dropped file.
FileNames Gets all selected or dropped files.

Methods

Method Description
GetExtensionsFromFilter() Returns the list of file extensions parsed from the Filter property.

Events

Event Description
FileDropped Occurs after one or more valid files have been selected or dropped onto the control.
Tip
The FileDroppedEventArgs class always provides a FileNames array, even when only a single file is selected. Use the OneFileDropped property to distinguish between single and multiple file selections.

Caution
Files that do not match the Filter property are ignored when dropped onto the control.
