cuiListbox

A list box with rounded corners, customizable item appearance, and hover highlighting.

Important Properties

Property Description
Rounding Controls the rounding of the list box itself.
ItemRounding Controls the rounding applied to individual items.
HoveredIndex Gets the index of the item currently under the mouse pointer, or -1 if none.
Tip
Since cuiListbox inherits from ListBox, you can use the standard Items, SelectedItem, SelectedIndex, and related members as usual.

Caution
This control always operates in single-selection mode (SelectionMode.One). Changing the SelectionMode property is not supported.
