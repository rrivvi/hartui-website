cuiLabel

An enhanced label control with flexible text layout and rendering options.

Important Properties

Property Description
Content The text displayed by the control.
HorizontalAlignment Controls the horizontal alignment of the text.
VerticalAlignment Controls the vertical alignment of the text.
WordWrap Enables wrapping text onto multiple lines. When disabled, the text is rendered on a single line.
AutoEllipsis Displays an ellipsis (...) when the text does not fit within the control and AutoSize is disabled.
AutoSize Automatically resizes the control to fit its content, taking the current font, padding, and WordWrap setting into account.
Trimming Specifies how clipped text is trimmed when AutoEllipsis is disabled.
Tip
Unlike the standard WinForms Label, this control supports vertical alignment, word wrapping, auto-sizing, trimming, and ellipsis.
