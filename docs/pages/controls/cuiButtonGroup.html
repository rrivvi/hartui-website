cuiButtonGroup

A customizable radio button that automatically unchecks other cuiButtonGroup controls belonging to the same group within the same parent container.

Important Properties

Property Description
Content The text displayed by the button. This property is also exposed through Text.
Group Determines which cuiButtonGroup controls behave as a single radio button group. Only controls with the same Group value affect each other.
Checked Gets or sets whether this button is currently selected. Changing this property raises the CheckedChanged event.
Image Displays an image next to the button text.
ImageExpand Adjusts the size of the displayed image. By default, the image is rendered using the current font height.
TextAlignment Aligns the button content to the left, center, or right using Near, Center, or Far.
TextSpacing Controls the spacing between the image and text.
TextPadding Adds padding around the button content. This property is most useful when TextAlignment is Near or Far.
Rounding Controls the corner radius of the button.
OutlineThickness Specifies the thickness of the button outline.

Methods

Method Description
PerformClick() Programmatically selects the button, updates the rest of its group, and raises the Click event.

Events

Event Description
CheckedChanged Occurs whenever the Checked property changes.
Tip
Like cuiButton, this control exposes separate colors for the background, outline, text, and image tint for each visual state.

Caution
Radio button behavior only applies to cuiButtonGroup controls that share the same parent container and the same Group value.
