cuiTextBox

A customizable text box supporting rounded corners, placeholder text, optional images, multiline input, and focus-specific appearance.

Important Properties

Property Description
Content Gets or sets the current text. This property is equivalent to Text.
PlaceholderText Text displayed when the control is empty and does not have focus.
Multiline Determines whether the control accepts multiple lines of text.
PasswordChar Displays password characters instead of the entered text.
UnderlinedStyle Draws only the bottom border instead of a complete outline.
Image Displays an optional image inside the text box. Its position, size, and tint can be customized.
Rounding Controls the corner radius of the text box.

Methods

Method Description
ResetText() Clears the current text and restores the placeholder if applicable.

Events

Event Description
ContentChanged Occurs whenever the text entered by the user changes.
Tip
Use the Content property instead of accessing the internal text box directly. Content and Text always refer to the same value.

Note
The placeholder text is only visible when the control is empty and does not have keyboard focus.
