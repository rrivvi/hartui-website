cuiFolderDropper

Accepts folders by drag and drop or, optionally, by opening a modern OpenFolderDialog when clicked.

Important Properties

Property Description
UploadWithClick Determines whether clicking the control opens an OpenFolderDialog.
Multiselect Allows the user to select or drop multiple folders at once.
FolderName Gets the first selected or dropped folder.
FolderNames Gets all selected or dropped folders.

Events

Event Description
FolderDropped Occurs after one or more valid folders have been selected or dropped onto the control.
Tip
The FolderDroppedEventArgs class always provides a FolderNames array, even when only a single folder is selected. Use the OneFolderDropped property to distinguish between single and multiple folder selections.

Caution
Files are ignored when dropped onto the control. Only existing folders are accepted.
