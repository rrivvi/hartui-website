cuiCalendarDatePicker

A button-like control that displays the currently selected date. Clicking the control opens the HartUI date picker dialog, allowing the user to select a new date.

Important Properties

Property Description
Content Gets or sets the selected date. The time portion is discarded, leaving only the date.
Theme Specifies whether the date picker dialog opens using the Light or Dark theme.
EnableThemeChangeButton Determines whether the dialog displays a button that allows the user to switch between light and dark themes.
PickerPosition Controls where the date picker dialog appears relative to the control.
ShowIcon Determines whether the calendar icon is displayed next to the selected date.
Icon Sets the image displayed beside the selected date.
IconTint Tints the displayed calendar icon.
Rounding Controls the corner radius of the control.

Methods

Method Description
ShowDialog() Opens the date picker dialog programmatically.

Events

Event Description
DateChanged Occurs whenever the selected date changes.
