cuiPictureBox

Displays an image with optional tinting, rotation, rounded corners, and an optional outline.

Important Properties

Property Description
Content The image displayed by the control.
ImageTint Applies a color tint to the displayed image.
Rotation Rotates the image in degrees.
Rounding Controls the corner radius of the control.
Tip
Image tinting is cached internally for performance. The cache is automatically regenerated when the Content or ImageTint property changes.
