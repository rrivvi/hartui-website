cuiSlider

A customizable horizontal slider for selecting a value within a specified range.

Important Properties

Property Description
Value Gets or sets the slider's current value. Changing it raises the ValueChanged event.
MinValue Specifies the minimum selectable value.
MaxValue Specifies the maximum selectable value.
ThumbOutlineThickness Controls the thickness of the thumb's outline.

Methods

Method Description
GetProgressPercentage() Returns the current slider position as a value between 0 and 1.

Events

Event Description
ValueChanged Occurs whenever the Value changes.
Caution
MinValue must be less than MaxValue. Invalid assignments are ignored.
