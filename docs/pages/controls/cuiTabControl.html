cuiTabControl

A custom tab control whose pages are created programmatically instead of through the designer. Each tab hosts a Control, such as a UserControl, and can optionally display an image.

Important Properties

Property Description
SelectedIndex Gets or changes the currently selected tab by index.
SelectedTab Gets or changes the currently selected TabPage.
Pages Provides a read-only collection of all tabs.
Count Gets the number of tabs currently in the control.
ShowPlus Determines whether the built-in add tab button is displayed.
ShowDelete Determines whether close buttons are shown on tab headers.
NamingConvention Specifies the base name used when automatically creating new tabs.
ScrollOffset Gets or sets the current horizontal scroll position of the tab strip.

Methods

Method Description
AddTab() Creates a new empty tab using the current NamingConvention.
AddTab(string) Creates a new empty tab with the specified title.
AddTab(string, Control) Creates a tab and automatically docks the supplied control to fill its contents.
AddTab(string, Bitmap, Control) Creates a tab with an image and automatically docks the supplied control.
SelectTab(int) Selects a tab by its index.
SelectTab(TabPage) Selects the specified tab.
RemoveTab(int) Removes the tab at the specified index.
RemoveTab(TabPage) Removes the specified tab.
RemoveSelectedTab() Removes the currently selected tab.
ClearTabs() Removes every tab from the control.
Contains(TabPage) Returns whether the specified tab belongs to this control.
IndexOf(TabPage) Returns the index of the specified tab.
GetUniqueTabName() Generates the next available tab name using NamingConvention.

Events

Event Description
TabAdded Occurs after a new tab has been created.
TabRemoved Occurs after a tab has been removed.
SelectedTabChanged Occurs when the selected tab changes.
Tip
The AddTab(..., Control) overloads automatically set the control's Dock property to DockStyle.Fill before adding it to the created tab.

Caution
Tabs cannot be created through the Windows Forms designer. They must be added programmatically using one of the AddTab() methods.
