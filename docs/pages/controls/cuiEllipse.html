cuiEllipse

Displays a simple ellipse with customizable fill color and optional outline.

Important Properties

Property Description
PanelColor Specifies the fill color of the ellipse.
OutlineColor Specifies the color of the ellipse outline.
OutlineThickness Controls the thickness of the outline in pixels.
Tip
Set OutlineColor to Color.Empty if you only want the filled ellipse without a visible border.

Note
The ellipse automatically resizes to fill the control while respecting the configured outline thickness.
