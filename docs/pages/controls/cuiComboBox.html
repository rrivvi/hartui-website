cuiComboBox

A modern combo box that displays its items in a separate rounded popup with a custom scrollbar when necessary.

Important Properties

Property Description
Items The collection of items displayed in the drop-down list.
SelectedIndex Gets or sets the index of the currently selected item. A value of -1 indicates that no item is selected.
SelectedItem Gets or sets the currently selected item by its text. Assigning an empty string clears the selection.
NoSelectionText The placeholder text displayed when SelectedIndex is -1.
SortAlphabetically Automatically sorts the items alphabetically before opening the drop-down.
MaxDropDownHeight Specifies the maximum height of the drop-down before scrolling is enabled.

Methods

Method Description
AddItem(string) Adds a new item to the combo box.
RemoveItem(string) Removes the first matching item from the combo box.

Events

Event Description
SelectedIndexChanged Occurs whenever the selected item changes.
Caution
When SortAlphabetically is enabled, the order of Items may change each time the drop-down is opened.

Note
If the currently selected item no longer exists after replacing the Items collection, the first available item is selected automatically.
