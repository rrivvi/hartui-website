cuiCircleProgressBar

The cuiCircleProgressBar is a ProgressBar in the shape of a circle.


Highlighted Properties

The control exposes more properties, these are the most important ones.

int

BorderWidth

Specifies how thick the border is.


int

ProgressValue

Specifies how much of the progress has been done. Its bounds are controlled by MinimumValue and MaximumValue


int

MaximumValue

The upper value limit. ProgressValue cannot be bigger than this number.


int

MinimumValue

The bottom value limit. ProgressValue cannot be smaller than this number.


bool

RoundedEnds

If True, renders the ends of the value bar rounded.
