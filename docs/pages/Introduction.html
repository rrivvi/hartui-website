Introduction


Welcome to the

HartUI documentation.

HartUI is a fork of the original CuoreUI library.
This project continues its development, fixes bugs, adds new controls and components, and keeps the library up to date.

Documentation

These pages focus on the features you'll use most often, such as important Properties, Methods, and usage notes.

Some self-explanatory properties (for example, many color customization properties) are intentionally omitted to keep the documentation concise.

Alert types

The following alert styles are used throughout the documentation:

Warning
Highlights behavior that may cause unexpected results or require special attention.

Caution
Points out limitations or things to be aware of before using a feature.

Tip
Provides useful information, recommendations, or shortcuts.

Note
Used for general information, additional details, or property lists.

That's all!
