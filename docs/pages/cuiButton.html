cuiButton

The cuiButton is a modern take on the button.
There's a similar control: cuiButtonGroup, which acts like a radio button.

Highlighted Properties

The control exposes many properties, these are the most important ones.

Padding

Rounding

Specifies how rounded the corners should be, in pixels.


String

Content

The text content of the control.
For newcomers, you can also use the Text variable in code, but it's not available from the Designer.


bool

Checked

If CheckButton == True, this property will determine if the button is in a checked state.
The checked state has priority above mouse states.

If Checked == true but CheckButton == false, the button is not in the checked state.


bool

CheckButton

Controls whether the control can become checked with a mouse click.


Image

Image

Your Image will be displayed to the left of the Content

Its Size can be manually adjusted with the ImageExpand property.
Its Position relative to the text Content can be adjusted with the TextSpacing property.
