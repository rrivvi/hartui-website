cuiCheckbox

The cuiCheckbox is a modern take on the check box.

Highlighted Properties

The control exposes many properties, these are the most important ones.

int

Rounding

Specifies how rounded the corners should be, in pixels.


bool

Checked

When set to true, the control will display in its checked state.
Otherwise, it will render normally as if it was unselected.


bool

ShowSymbols

Controls whether the checkmark symbol appears when Checked == true
(or an "X" cross symbol, in case of Checked == false)
