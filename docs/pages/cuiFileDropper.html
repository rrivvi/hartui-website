cuiFileDropper

The is a custom control allowing the user to drop a file (or multiple files) onto it.
The user can also just click it to open an OpenFileDialog.

You, as a developer can attach to the FileDropped event:
It provides the filenames through its custom FileDroppedEventArgs for ease of use.
Alternatively, you can use the FileName or FileNames getter properties.

There's a similar control: cuiFolderDropper, which lets you drop/select folders.

Highlighted Properties

The control exposes many properties, these are the most important ones.

Padding

Rounding

Specifies how rounded the corners should be, in pixels.


bool

Multiselect

If true, the user is able to drop/select multiple files.


String

Filter

You, the developer, can limit what extensions the user can select with this property.
It works similar to the OpenFileDialog's Filter property.

Here's an example: PNG Files|*.png|WEBP Files|*.webp
The filter above will let the user choose either .png or .webp files.


String

NormalContent

The text content when the user is not doing anything with the control.


String

HoverContent

The text content when the user is dragging something over the control, but still hasn't released their mouse button.


bool

UploadWithClick

Controls whether the user will be able to click on the control to choose a file (or files) with an OpenFileDialog.
If false, the UploadContent text won't be rendered.


String

UploadContent

The text content telling the user they can also click to select.
This text will display both when the control is idle, but also when dropping a file.


Image

Image

This Image will display above the NormalContent text.
Its spacing and size can be controlled with the ImageSize and ImagePadding respectively.
