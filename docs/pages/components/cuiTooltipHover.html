cuiTooltipHover

Displays a customizable tooltip when the mouse pointer hovers over a target control. The tooltip follows the cursor until it leaves the control.

Important Properties

Property Description
TargetControl The control that displays the tooltip when hovered.
Content The text displayed inside the tooltip.
TooltipPosition Determines where the tooltip is positioned relative to the mouse cursor. Available values are Top, Bottom, Left, Right, and Custom.
TooltipPositionOffset Applies an additional position offset. This is primarily useful when TooltipPosition is set to Custom.
ForeColor Sets the tooltip text color.
BackColor Sets the tooltip background color.
Tip
Use TooltipPositionOffset to fine-tune the tooltip location without changing its general placement.

Caution
Only one HartUI tooltip can be visible at a time. All cuiTooltipHover components share the same tooltip window and appearance.

Note
The tooltip automatically follows the mouse cursor while it remains over the target control and is hidden as soon as the cursor leaves it.
