cuiFormRounder

Adds rounded corners to borderless forms by creating and managing a companion window that renders the rounded outline. This component is intended for Windows versions that do not provide native rounded window corners.

Important Properties

Property Description
TargetForm The form that should receive rounded corners. Only one cuiFormRounder can be assigned to a form at a time.
Rounding Specifies the radius of the form's rounded corners. Setting this property updates the form immediately.
OutlineColor Specifies the color of the outline drawn around the rounded form.

Methods

Method Description
UpdateRoundedFormRegion() Rebuilds the companion window's region to match the current size and shape of the target form.
Warning
Only one cuiFormRounder may be attached to a form. Attempting to assign a second instance to the same form will result in an exception.

Caution
The target form's FormBorderStyle is automatically set to None when the component initializes.

Tip
Pair this component with cuiResizeGrip if you want a borderless form that can still be resized by the user.

Note
Other HartUI components, such as cuiMessageDialog and cuiFormHideCaptureScreenshot, automatically detect and integrate with cuiFormRounder when it is attached to the target form.
