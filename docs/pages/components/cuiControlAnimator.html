cuiControlAnimator

Animates a control's position and/or opacity using configurable easing functions. The animation can start automatically when the control is first displayed, or manually by calling PlayAnimation().

Important Properties

Property Description
TargetControl The control to animate. Forms are not supported; use cuiFormAnimator instead.
TargetLocation The destination location when AnimateLocation is enabled.
AnimateLocation Determines whether the control moves to TargetLocation during the animation.
AnimateOpacity Animates the control between fully visible and fully transparent by drawing an overlay during painting.
TargetOpacity Specifies whether the animation should end with the control visible or transparent.
Duration Length of the animation in milliseconds.
EasingType Selects the easing function used for the animation.
AnimateOnStart Automatically starts the animation after the target control's handle has been created.

Methods

Method Description
PlayAnimation() Starts the animation. If another animation is already running, the call is ignored.
IsAnimationFinished() Returns true when no animation is currently in progress.

Events

Event Description
AnimationStarted Raised when an animation begins.
AnimationEnded Raised after the animation has completed.
Caution
TargetLocation is only used when AnimateLocation is enabled. If location animation is disabled, the control remains in its current position.

Tip
The same animator instance can be reused by assigning a different TargetControl before calling PlayAnimation().

Note
Opacity animation is implemented by attaching to the target control's Paint event for the duration of the animation. Once the animation finishes, the event handler is removed automatically.
