cuiControlAnimator

Animates a control's position and/or opacity using configurable easing functions. Animations can start automatically when the target control is first shown or manually by calling PlayAnimation().

Important Properties

Property Description
TargetControl The control to animate. Forms are not supported; use cuiFormAnimator instead.
TargetLocation The destination location when AnimateLocation is enabled.
AnimateLocation Determines whether the control moves to TargetLocation during the animation.
AnimateOpacity Animates the control between fully visible and fully transparent by drawing an overlay during painting.
TargetOpacity Specifies whether the animation should end with the control visible or transparent.
Duration Length of the animation in milliseconds.
EasingType Selects the easing function used for the animation.
AnimateOnStart Automatically starts the animation when the target control is first created.
IsAnimating Indicates whether an animation is currently running.

Methods

Method Description
PlayAnimation() Starts the animation. If another animation is already running, the call is ignored.
CancelAnimation() Immediately cancels the current animation, leaving the control at its current state.

Events

Event Description
AnimationStarted Raised when an animation begins.
AnimationEnded Raised after an animation completes normally.
Tip
This component supports animating controls with Anchor values like Top | Right. If the target control is moved by other code while a location animation is in progress, the animator automatically preserves that movement and continues animating relative to the new position.

Caution
TargetLocation is only used when AnimateLocation is enabled. If location animation is disabled, the control remains in its current position.

Note
Changing TargetControl automatically cancels any active animation. Opacity animation is implemented by temporarily attaching to the control's Paint event and is cleaned up automatically when the animation finishes or is cancelled.
