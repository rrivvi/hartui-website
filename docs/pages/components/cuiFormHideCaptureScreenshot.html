cuiFormHideCaptureScreenshot

Prevents a form from appearing in screenshots and screen recordings by applying a Windows display affinity setting. The exclusion mode can be configured to either hide the window completely or replace it with a black rectangle.

Important Properties

Property Description
TargetForm The form to which the capture exclusion should be applied.
ExclusionType Determines how the form appears in screenshots and screen recordings:
  • Disappear - The window is omitted from the captured image.
  • Black - The window is replaced with a black rectangle.
  • None - Disables capture protection.

Methods

Method Description
RefreshExclusionType() Reapplies the current ExclusionType to the target form.
Tip
Changing ExclusionType automatically applies the new setting. Calling RefreshExclusionType() is only necessary if the protection needs to be reapplied manually.

Note
When used together with cuiFormRounder, the exclusion setting is automatically applied to both the original form and its rounded host window.
