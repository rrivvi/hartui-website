cuiControlBlur

Applies a blur effect to an existing control by intercepting its paint operation. The blurred image is cached and automatically regenerated whenever the control's contents change.

Important Properties

Property Description
TargetControl The control that should be blurred. Forms are not supported.
BlurAmount Controls the strength of the blur. Larger values produce a stronger blur effect.
Warning
This component replaces the target control's rendered appearance with a blurred version. Any changes to the control's contents automatically invalidate the cached image and cause the blur to be regenerated.

Tip
The blurred image is cached for performance. It is only recreated when the target control is invalidated, resized, or when BlurAmount changes.
