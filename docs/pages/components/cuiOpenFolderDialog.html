cuiOpenFolderDialog

Provides access to the modern Windows folder selection dialog as an alternative to the legacy FolderBrowserDialog. Supports custom labels and selecting one or multiple folders.

Important Properties

Property Description
Title Sets the dialog title. Leave empty to use the default system text.
OkButtonLabel Changes the text displayed on the confirmation button. Leave empty to use the default system text.
Multiselect Allows the user to select multiple folders instead of a single folder.
FolderName Contains the first selected folder path after a successful selection.
FolderNames Contains all selected folder paths after a successful selection when Multiselect is enabled.

Methods

Method Description
ShowDialog() Opens the folder selection dialog and returns the user's selection result.
Tip
When Multiselect is enabled, use FolderNames instead of FolderName to access all selected folders.

Note
FolderName and FolderNames are updated only after the dialog closes with a successful selection. They retain their previous values if the user cancels the dialog.
