cuiMessageDialog

Displays a modern customizable message dialog while preventing interaction with the parent form until the user selects an option. The dialog supports common MessageBoxButtons layouts and returns a standard DialogResult.

Important Properties

Property Description
DimColor The color and opacity of the overlay applied behind the dialog while it is open.
DialogSize The default size of the dialog window. The height can automatically increase when the displayed text requires additional space.
DialogPadding The padding applied between the dialog content and its edges.
ButtonSize Controls the size of the buttons displayed inside the dialog.
Rounding Controls the corner radius of the dialog window.
BackColor The dialog background color.
ForeColor The dialog text color.
OKText The text displayed on the OK button.
YesText The text displayed on the Yes button.
NoText The text displayed on the No button.
CancelText The text displayed on the Cancel button.

Methods

Method Description
ShowDialog(Form parentForm, string description, string title = "") Displays a dialog containing the specified message using an OK button.
ShowDialog(Form parentForm, string description, string title, MessageBoxButtons messageBoxButtons) Displays a dialog using the specified button layout and returns the user's selection.
Tip
The dialog uses the standard DialogResult values, making it possible to handle responses in the same way as a regular WinForms message box.

Note
The dialog instance is created internally each time ShowDialog() is called. Configure the component properties before displaying the dialog.
