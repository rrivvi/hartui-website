cuiControlDrag

Allows a form to be moved by dragging a control instead of the form's title bar. This is primarily intended for borderless windows with custom title bars.

Important Properties

Property Description
TargetControl The control that acts as the drag surface. Dragging this control with the left mouse button moves its parent form.
Tip
This component automatically locates the parent form using FindForm(). No additional configuration is required after assigning TargetControl.

Note
Assigning a different TargetControl automatically detaches the component from the previous control and attaches it to the new one.
