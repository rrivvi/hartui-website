cuiFormAnimator

Animates a form's Opacity property using configurable easing functions. The animation can play automatically when the form is first shown or be started manually.

Important Properties

Property Description
TargetForm The form whose opacity will be animated.
AnimateOnStart Automatically plays the animation the first time the target form is shown.
StartOpacity The opacity value at the beginning of the animation. Values are automatically clamped between 0 and 1.
TargetOpacity The opacity value when the animation finishes. Values are automatically clamped between 0 and 1.
Duration Length of the animation in milliseconds.
EasingType Selects the easing function used during the opacity transition.

Methods

Method Description
PlayAnimation() Starts the opacity animation manually.
Caution
This component only animates the form's Opacity. It does not animate the form's position or size.

Tip
Set AnimateOnStart to false if you want to control exactly when the animation begins by calling PlayAnimation().
