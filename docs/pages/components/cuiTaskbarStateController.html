cuiTaskbarStateController

Controls the appearance of the application's taskbar button by displaying a progress indicator or changing its state to indicate paused or failed operations.

Important Properties

Property Description
TargetForm The form whose taskbar button will be updated.
State Selects the taskbar progress state. Available values are Progress, Paused, Error, and Default.
ProgressValue The current progress value displayed when the State is not Default.
MaxValue Specifies the maximum progress value used to calculate the taskbar progress percentage.

Methods

Method Description
Flash() Continuously flashes the application's taskbar button until the window receives focus, helping draw the user's attention.
Tip
Updating ProgressValue, MaxValue, or State immediately refreshes the taskbar progress indicator.

Caution
When State is set to Default, the taskbar progress indicator is completely removed, regardless of the current ProgressValue.

Note
The taskbar progress indicator and flashing behavior are only available on supported versions of Windows.
