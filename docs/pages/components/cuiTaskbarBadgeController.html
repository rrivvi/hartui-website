cuiTaskbarBadgeController

Displays an overlay badge on the application's taskbar icon. The badge can either be a custom Image or an automatically generated numeric notification indicator.

Important Properties

Property Description
TargetForm The form whose taskbar icon will display the badge.
Image Sets a custom overlay icon. Images are automatically resized to 16×16 if necessary.
Description The accessibility description associated with the overlay icon.
NumericValue Displays a numeric badge instead of the custom image when greater than 0. Values above 9 are displayed as 9+.
NumericBackColor Background color of the generated numeric badge.
NumericForeColor Text color of the generated numeric badge.
NumericFont Font used when drawing the numeric badge.
Tip
Setting NumericValue to a value greater than 0 temporarily overrides the Image property. Setting it back to 0 restores the custom image if one has been assigned.

Caution
Overlay icons are only supported on compatible versions of Windows. On unsupported systems, the component performs no action.

Note
If no custom image is assigned and NumericValue is 0 or less, any existing taskbar overlay is removed.
