cuiFolderDropper

The is a custom control allowing the user to drop a folder (or multiple folders) onto it.
The user can also just click it to open an OpenFolderDialog
(CuoreUI implements it for older versions of .NET Framework).

You, as a developer can attach to the FolderDropped event:
It provides the folder paths through its custom FolderDroppedEventArgs for ease of use.
Alternatively, you can use the FolderName or FolderNames getter properties.

There's a similar control: cuiFileDropper, which lets you drop/select files.

Highlighted Properties

The control exposes many properties, these are the most important ones.

Padding

Rounding

Specifies how rounded the corners should be, in pixels.


bool

Multiselect

If true, the user is able to drop/select multiple folders.


String

NormalContent

The text content when the user is not doing anything with the control.


String

HoverContent

The text content when the user is dragging something over the control, but still hasn't released their mouse button.


bool

UploadWithClick

Controls whether the user will be able to click on the control to choose a folder (or folders) with an OpenFolderDialog.
If false, the UploadContent text won't be rendered.


String

UploadContent

The text content telling the user they can also click to select.
This text will display both when the control is idle, but also when dropping a folder.


Image

Image

This Image will display above the NormalContent text.
Its spacing and size can be controlled with the ImageSize and ImagePadding respectively.
